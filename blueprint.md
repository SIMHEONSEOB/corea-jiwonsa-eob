# 프로젝트 개요

이 프로젝트는 경기도 청년 지원 사업 정보를 웹 페이지에 표시합니다. 현대적인 웹 표준과 웹 컴포넌트를 활용하여 재사용 가능하고 관리하기 쉬운 UI를 구현합니다.

## 스타일, 디자인 및 기능

*   **반응형 디자인:** 다양한 화면 크기에 맞춰 레이아웃이 조정됩니다.
*   **카드형 UI:** 지원 사업 정보는 카드 형태로 표시되어 시각적으로 구분됩니다.
*   **CSS 변수:** `:root`에 정의된 CSS 변수를 사용하여 색상, 그림자 등의 스타일을 중앙에서 관리하고 쉽게 변경할 수 있습니다.
*   **단일 카드:** 현재는 "경기도 청년 면접수당"에 대한 단일 카드만 표시됩니다.

## 현재 변경 요청에 대한 계획 및 단계

**요청:**
1.  제목을 "경기도 청년 지원 사업"으로 변경.
2.  부제목 "전국민 지원 사업"을 "청년 지원 사업"으로 변경.
3.  기존의 3개 카드를 모두 삭제하고 "경기도 청년 면접수당" 정보를 담은 새 카드로 교체.
4.  변경 사항 배포.

**설명:**
사용자님의 요청에 따라 웹 페이지의 구조와 데이터를 완전히 개편했습니다.

**변경 내용:**

1.  **`index.html` 변경:**
    *   페이지 제목 (`<title>`)을 "경기도 청년 지원 사업"으로 변경했습니다.
    *   메인 헤딩 (`<h1>`)을 "경기도 청년 지원 사업"으로 변경했습니다.
    *   부제목 (`<h2>`)을 "청년 지원 사업"으로 변경했습니다.
    *   기존의 카드 컨테이너 내용을 "경기도 청년 면접수당" 정보를 담은 새로운 카드로 교체했습니다.

2.  **`style.css` 변경:**
    *   `.card` 클래스에 대한 기본 스타일을 추가하여 카드가 시각적으로 보기 좋게 만들었습니다.
    *   `#policy-list h2` 셀렉터를 `#national-support-section h2`로 변경하여 HTML 구조와 일치시켰습니다.

**결론:**
이제 웹 페이지에 접속하면 "경기도 청년 지원 사업"이라는 새로운 제목과 함께 "청년 지원 사업" 섹션 아래에 "경기도 청년 면접수당"에 대한 단일 카드가 표시될 것입니다.

**확인 지침:**
*   **브라우저 캐시를 지우고 페이지를 새로고침**하거나, 개발 서버를 사용 중이라면 **개발 서버를 재시작**해 보세요.
*   웹 페이지에서 변경된 제목, 부제목, 그리고 새로 추가된 "경기도 청년 면접수당" 카드가 올바르게 표시되는지 확인해 주세요.

이상의 확인 후 결과를 알려주시면 감사하겠습니다. 모든 문제가 해결되었다면, 요청하신 대로 최종 배포(git push)를 진행하겠습니다.
