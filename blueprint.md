# 국가 지원 사업 정보 사이트 블루프린트

## 개요

이 프로젝트는 사용자에게 다양한 국가 지원 사업에 대한 정보를 제공하는 것을 목표로 하는 웹 애플리케이션입니다. 사용자는 이 사이트를 통해 최신 지원 사업을 쉽게 찾고, 주요 정보를 확인할 수 있습니다.

## 디자인 및 스타일

- **모던하고 깔끔한 디자인:** 최신 웹 표준을 준수하며, 사용자가 정보에 쉽게 집중할 수 있도록 깔끔하고 직관적인 UI를 제공합니다.
- **반응형 레이아웃:** 모바일, 태블릿, 데스크톱 등 다양한 디바이스에서 최적의 경험을 제공합니다.
- **인터랙티브 요소:** 동적인 UI 요소와 시각 효과를 통해 사용자 참여를 유도합니다.
- **웹 컴포넌트:** 재사용 가능한 UI 컴포넌트를 활용하여 코드의 일관성과 유지보수성을 높입니다.

## 기능

- **지원 사업 목록:** 현재 진행 중인 다양한 국가 지원 사업 목록을 표시합니다.
- **상세 정보:** 각 지원 사업에 대한 상세 정보를 제공합니다.
- **검색 및 필터링:** 사용자가 원하는 지원 사업을 쉽게 찾을 수 있도록 검색 및 필터링 기능을 제공합니다.

## 현재 구현 계획

1. **`index.html` 구조 변경:**
    - 페이지 제목을 "국가 지원 사업 정보"로 변경합니다.
    - 헤더, 메인 콘텐츠, 푸터 영역으로 구조를 나눕니다.
    - 지원 사업 정보를 표시할 컨테이너를 추가합니다.
2. **`style.css` 스타일링:**
    - 모던하고 시각적으로 매력적인 스타일을 적용합니다.
    - CSS 변수, Flexbox, Grid를 사용하여 유연하고 반응형 레이아웃을 구현합니다.
3. **`main.js` 기능 구현:**
    - 웹 컴포넌트를 정의하여 지원 사업 카드를 생성합니다.
    - 동적으로 지원 사업 데이터를 가져와 화면에 렌더링합니다.
4. **'예비창업패키지' 정보 추가:**
    - `main.js`의 `policyData` 배열에 '예비창업패키지' 정보를 추가하여 화면에 표시합니다.
